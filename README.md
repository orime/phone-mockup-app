# 手机截图美化器

这是一个基于React的应用程序，用于美化手机截图。您可以上传手机截图，选择不同的手机型号外壳，调整背景颜色，并将最终结果导出为图片或复制到剪贴板。

## 功能特点

- 支持上传任意手机截图
- 提供多种手机外壳样式（iPhone 14、iPhone 15、安卓）
- 自定义背景颜色
- 导出为PNG图片
- 一键复制到剪贴板

## 技术栈

- React 19
- TypeScript
- Vite
- react-colorful（颜色选择器）
- html-to-image（图片导出）
- file-saver（文件下载）

## 本地开发

### 安装依赖

```bash
pnpm install
```

### 启动开发服务器

```bash
pnpm dev
```

### 构建生产版本

```bash
pnpm build
```

## 使用指南

1. 上传手机截图
2. 选择喜欢的手机型号
3. 调整背景颜色
4. 导出或复制最终效果图
